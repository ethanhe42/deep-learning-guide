CS291K - Project Ideas  
No. Topics Notes
1. Word Spelling Correction Use neural network to check the spelling correctness in articles.
2. Word pronunciation prediction Use neural network to predict how to pronounce a new word or a new name
3. Produce new recipes Use neural network to generate new recipes
4. Write a new poem Use neural network to generate new poems
5. Automatically write SQL statements Given a user natural language description, automatically generate an SQL statements
6. Table Extraction In plain text documents, there could be many tables, how to extract these tables; you can view each row in a table as a data point, you want to find the commonality of these rows and extract them.
7. Command understanding Given a user's command, automatically generate a control sequence.  For example, "Please turn on the oven at 5:00pm, cook it for 1 hour and then warm"
8. Ticket Booking/Changing/Cancelling Write a neural network program to automatically handle trip booking commands.
9. Restaurant  reservation / online ordering Using Google API to convert the speech to text, and then develop neural network to convert text to orders.  You can sell this program if it works very well.
10. Word paraphrasing I wrote a sentence and would like to beautify  it .  Can you write a neural network program to recommend a few words to replace some words in this sentence?  You can get many beautiful sentences from the Web and news articles.
11. Similar questions Can you write a a neural network program to detect two questions are similar questions?  Think about the killer application in Stackoverflow and Piazza.
12. Chatbot Write a neural network program that can chat with a person, like Facebook's future Chatbot.
13. Question answer Write a neural network program that can answer simple questions, e.g., who is the president of the United States
14. Arithmetic operators Train a neural network that can do arithmetic operations
15. Learn rules from neural networks Given a neural network, how to infer some rules hidden in the network so that we can easily understand the knowledge hidden in this network.
16. (new) Predict the sales of T-shirt Given a set of T-shirt images and other properties, predict whether they will be sold well or not.  (A real T-shirt retailer dataset will be provided).
